# Portfolio — Shanmukha Sai Praneeth

Next.js Vercel Project

This is a web application built using Next.js and deployed on Vercel.

* Next.js
* React
* Node.js
* Vercel (for deployment)



Follow these steps to run the project locally:

1. Clone the repository
git clone https://github.com/OmniShenron/Next.jsVercel.git
cd Next.jsVercel



1. Install dependencies
npm install
2. Run the development server
npm run dev
3. Open in browser
http://localhost:3000
